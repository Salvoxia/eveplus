THEME INSTALLATION 

Move the folder "EvePlus" to your killboard url/themes folder.


SHIP FITTING DISPLAY 

Move the folder "panel" to your killboard url/img folder. When prompted to "Overwrite" select yes.
- On your killboard, go to Admin > Kill Details > Fitting Panel: and select "tyrannis" as the panel theme.


BANNER 

Move your banner image to the killboard url/banner folder. To change the current banner go to Admin > Global Options > Global Look: Banner.
The ideal width when creating a banner for this theme is: 1200px or below.
